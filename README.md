<br/></br>### 课程介绍:
出发点不是要批判C语言，而是要帮助C程序员绕过编程过程中的陷阱和障碍，分为8章，分别从词法分析、语法语义、连接、库函数、预处理器、可移植性缺陷等几个方面分析了C编程中可能遇到的问题。结尾，作者用一章的篇幅给出了若干具有实用价值的建议。
